# Uber
This project is a console-based C++ simulation of a ride-sharing system similar to Uber. The program allows users to create a city grid, add drivers, place obstacles, set pickup and destination points, and find the best driver for a ride request.
The system uses Breadth-First Search (BFS) to calculate the shortest path between locations while avoiding blocked cells. It also includes a Dynamic Programming (DP) module for optimizing assignments between multiple drivers and ride requests.
The project demonstrates important computer science concepts such as Object-Oriented Programming (OOP), graph traversal, shortest path algorithms, dynamic programming, and modular software design.
